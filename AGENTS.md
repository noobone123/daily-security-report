# Daily Security Digest Repo Notes

This repository packages the `daily-security-digest` skill and its supporting subagents.

If the skill has been installed for the current user, prefer using the installed `daily-security-digest` skill instead of reinterpreting the workflow from scratch.

The shared runtime lives under `skills/daily-security-digest/`.
The Codex subagents live under `.codex/agents/`.
The Claude Code subagents live under `agents/`.
