# Topics

## Browser Exploitation

### Care About

Browser exploitation research, renderer bugs, sandbox escapes, exploit writeups, and mitigation bypasses.

### Usually Ignore

Product marketing, unrelated browser extensions, and shallow reposts with no technical content.

### Reporting Angle

Highlight technical novelty, root cause detail, exploitability, and whether the material contains a useful writeup or proof of concept.

## Cloud Detection

### Care About

Cloud detection engineering, detections-as-code, infrastructure telemetry, rules, and defensive playbooks.

### Usually Ignore

Generic cloud marketing, pricing posts, and shallow observability content with no detection value.

### Reporting Angle

Call out practical detection ideas, reusable rules, and what defenders can operationalize quickly.
